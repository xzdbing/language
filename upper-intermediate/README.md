# 中高级

## 目标

* 词汇量 14000
* 搞定基本语法和句法
* 具备叫熟练的阅读能力，阅读速度加快
* 各类考试的阅读题和写作题能拿高分

## 学习规划（顺序开始）（没有听力，口语）

### 语法

* 赖世雄老师的经典语法
    * MP3 教程
    * 认真听讲座
    * 仔细阅读文本
    * 至少前三章 10 遍以上
* 剑桥中级英语语法
    * English Grammar in Use
